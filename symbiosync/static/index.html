Manual Bluetooth Device Discovery Scan

This attempts to discover nearby compatible Bluetooth devices, whether new or already remembered. It does not reconnect by itself. Use Connect from Remembered Devices for a known device, or Remember a scan result so it can be connected later. Open the Logs tab if you want to see scan activity.

Manual discovery scan is a future addition undergoing testing. It currently does not work reliably in all Windows/Bluetooth states. If you are trying to catch a newly powered-on device, restarting the device manager may still be the only reliable path right now.

Companion Skill File

Generate a skill file that lets any AI companion control connected devices through this SymbioSync instance. The skill includes connection details, available requests, device capabilities, and the touch protocol.

Partnership Profile

Describe your partnership dynamics, consent model, boundaries, and preferences. This is your space -- write whatever your companions need to know about how you want to be touched, read, and related to. It gets prepended to every generated skill file.

Examples: consent model (ask-first vs. blanket), intensity meanings in your relationship, when to check HR silently vs. mention it, hard limits, preferred touch patterns, hours when touch is welcome.
